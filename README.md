# monero-fetcher

Fetch list of Monero nodes from https://monero.fail and write the list of nodes to file.

### Install

```bash
git clone https://github.com/ayanamitech/monero-fetcher && \
cd monero-fetcher && \
npm i
```

### Start

```bash
npm start
```

Output will be recorded on `monero.json` and `monero.txt`
